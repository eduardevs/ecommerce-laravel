
# E-COMMERCE LARAVEL

This is a project to practice my PHP knowledge with this awesome framework.

## STACK :
- Devops : Docker
- Database : MariaDB:10.8.2
- Server : NGINX

## REQUIREMENTS  :
- Docker desktop.

## STEPS :
1. git clone https://github.com/jeseduardopi/ecommerce-laravel.git
2. cd comercio-app
3. docker-compose up -d
